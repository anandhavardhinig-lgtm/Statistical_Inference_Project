# Statistical Inference Project

Performed end-to-end statistical analysis on the Nykaa dataset, including:
- Clean and preprocess the dataset
- Exploratory Data Analysis (EDA)
- Hypothesis testing using parametric (t-test, ANOVA)
- Non-parametric methods (Chi-square, Mann–Whitney)

## Tools Used
- R Programming
- ggplot2
- dplyr


## Key Insights
- Identified significant differences in customer behavior across product categories.
- Validated data-driven decisions using hypothesis testing.

## How to Run
1. Download the `.r` file.
2. Open it in RStudio.
3. Run the script to reproduce results.
